# GPT Image Installer Prompts

These source images are intended to be generated with GPT Image and saved as:

- `installer/assets/source/wizard-main-gpt-image.png`
- `installer/assets/source/wizard-small-gpt-image.png`

## Main Wizard Image

Create a calm, minimal abstract system-utility artwork for a Windows installer.
Soft warm-neutral background, airy composition, geometric layers, subtle pulse-wave motif,
single restrained teal accent, no text, no devices, no people, no glossy marketing look,
clean premium desktop-software feeling, lightweight and quiet.

## Small Wizard Image

Create a compact companion artwork for a Windows installer sidebar.
Minimal abstract geometry, soft neutral field, pulse-wave motif, restrained teal accent,
same visual language as the main image, no text, no people, no product mockups,
clean and quiet.
